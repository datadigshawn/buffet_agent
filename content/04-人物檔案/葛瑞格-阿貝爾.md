# 葛瑞格·阿貝爾(Greg Abel)

## 一句話介紹
**巴菲特親自指定的下一任 CEO——一個從加拿大愛德蒙頓的工人家庭走到伯克希爾頂端的人。**

## 背景軌跡

| 年份 | 經歷 |
|------|------|
| 1962 | 生於加拿大亞伯達省愛德蒙頓 |
| 1984 | 阿爾伯塔大學商學院畢業,主修會計 |
| 1980s 末 | 普華永道(會計師事務所)職涯起點 |
| 1992 | 加入 CalEnergy(後來的 MidAmerican Energy) |
| 1999 | 伯克希爾收購 MidAmerican Energy(後改名 [[伯克希爾能源]]) |

| 2018 | **晉升伯克希爾副董事長,主管非保險業務** |
| **2021 年 5 月** | 蒙格在年會中「不小心」提到「**葛瑞格會接 CEO**」,巴菲特隨後正式確認 |

## 阿貝爾的核心能力:資本配置型營運者

### 一、能源與公用事業的長線思維
公用事業生意以 **十年起跳**為單位思考——
電廠、輸電網、天然氣管線都是 30-50 年的資產。
阿貝爾在這個生態系內鍛鍊出的時間視野,**剛好和巴菲特的長期主義同頻**。

### 二、再投資紀律
BHE 是伯克希爾體系內 **再投資強度最高**的子公司之一——
每年數十億美元的資本支出,都要過阿貝爾的關。
這個過程中累積的判斷力,**正是 CEO 該有的核心肌群**。

### 三、不是選股人,是經營者
巴菲特刻意強調:**未來的投資組合不會由阿貝爾主導**,而會由 [[陶德-康姆斯]]、[[泰德-韋施勒]] 與 [[阿吉特-賈恩]] 在不同領域各司其職。
**阿貝爾要做的是 CEO,不是投資長**。
這個分工比預期的更清楚,也更健康。

## 接班結構

```
巴菲特(現任 CEO/Chairman)
   │
   ├── Greg Abel ─── 非保險業務副董事長 → 下一任 CEO
   ├── Ajit Jain ─── 保險業務副董事長(獨立分管,不變)
   ├── Howard Buffett ─ 非執行董事長(僅守護文化,不參與營運)
   ├── Todd Combs / Ted Weschler ─ 投資組合管理
```

這個結構意味著:**伯克希爾不是「找一個巴菲特接班人」,而是「把巴菲特這份工作拆給三個人做」**。
這是巴菲特送給股東的最後一份禮物。
