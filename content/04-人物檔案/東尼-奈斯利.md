# 東尼·奈斯利(Tony Nicely)

## 一句話介紹
**18 歲進 GEICO,57 年後以 CEO 身份退休——巴菲特用他來定義「什麼叫一個偉大的經理人」。**

## 背景軌跡

| 年份 | 經歷 |
|------|------|
| 1943 | 生於美國華盛頓特區 |
| **1961**(18 歲) | 加入 GEICO,從基層員工做起 |
| 1980s | 一路晉升,GEICO 此時陷入 70 年代末的經營危機 |
| **1993** | 接任 GEICO CEO |
| 1996 | 伯克希爾全資收購 GEICO,奈斯利留任 CEO |
| 1993-2018 | **任內 GEICO 市佔率從不到 2% 成長到約 13%**——汽車保險業的史詩級成長故事 |
| 2018 | 退休,Bill Roberts 短暫接任,後由 [[陶德-康姆斯]] 接手 |

## 奈斯利的功績

### 一、抓住直銷模式的時間窗
傳統汽車保險靠保險經紀人收佣金;
奈斯利讓 GEICO **押注直銷 + 大規模廣告**,結構性地把成本壓低,反映在保費上,形成正向循環:
**便宜保費 → 更多客戶 → 規模經濟 → 更便宜保費**。
這是 [[護城河]] 中「成本優勢」型的教科書範例。

### 二、廣告策略
奈斯利任內塑造了 GEICO 那隻**會講話的小蜥蜴**(Gecko)以及一系列幽默廣告,讓 GEICO 在消費者心智佔有率上超越許多老牌對手。

### 三、紀律不被規模沖昏頭
GEICO 在他手上規模放大數倍,但 **承保紀律幾乎沒有走樣**——
這在快速成長的保險公司裡極為罕見(失去紀律的成長,結局通常是大虧損)。

## 為何巴菲特反覆在信中讚美他

巴菲特把奈斯利視為 **「給對人就放手」的最佳範例**——
從 1996 年伯克希爾全資收購到 2018 年退休,**巴菲特幾乎沒有干預過 GEICO 的營運**。
而 GEICO 在這段期間貢獻給伯克希爾的承保利潤 + [[浮存金]],累積金額無法估算。

這就是巴菲特口中:**「我能做的最有價值的工作,是把不該由我做的事交出去。」**
