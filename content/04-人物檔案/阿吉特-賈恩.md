# 阿吉特·賈恩(Ajit Jain)

## 一句話介紹
**伯克希爾「無法被取代」的人——巴菲特公開說「如果讓我和他換位置,我會立刻同意」。**

## 背景軌跡

| 時期 | 經歷 |
|------|------|
| 1951 | 生於印度奧里薩邦 |
| 1972 | 印度理工學院(IIT 卡拉格普爾)機械工程學位 |
| 1973-78 | IBM 印度業務 |
| 1978 | 哈佛商學院 MBA |
| 1980s 初 | McKinsey 顧問 |
| **1986** | 加入伯克希爾旗下國民賠償保險(National Indemnity) |
| 1990s | 建立伯克希爾的「巨災再保險」業務,從零做到世界級 |
| 2018 | **晉升為保險業務副董事長**(同時 [[葛瑞格-阿貝爾]] 接非保險業務副董事長) |
| 2021 後 | 在巴菲特退休後將繼續主管保險業務(分工已明確) |

## 為何巴菲特對阿吉特推崇備至

### 一、超人類的工作節奏與深度
巴菲特多次以開玩笑方式描述:
**阿吉特一個人就是一支保險團隊**——
他能在電話裡判斷一張數十億美元的再保險合約該不該接,且勝率極高。

### 二、定價權建立在判斷力上,而不是公式
傳統再保險仰賴精算模型;
阿吉特的競爭力是 **能在沒有歷史數據的情境下定價**——
911、2005 卡崔娜風災、新興巨災,他都接得下來,且賺得到錢。

### 三、複利型的資本貢獻
他經營的保險業務,源源不絕地產出 [[浮存金]]——
**這就是伯克希爾投資組合最關鍵的低成本資金來源**。
換言之:阿吉特等於是巴菲特投資的「彈藥供應官」。

## 巴菲特對阿吉特的標誌性短句(≤15 字)

- 「拿我跟阿吉特交換,我立刻答應。」(語意改寫,多次出現)
- 「他每天都在創造價值。」
- 「他無法被複製,也無法被替代。」

## 與其他關鍵人物的關係

- 與 [[查理-蒙格]]:蒙格也多次表達對阿吉特的高度評價,稱其為「上帝的恩賜」(語意改寫)。
