# 馬蒙集團(Marmon Group)

## 一句話定位
**Pritzker 家族(凱悅酒店家族)的工業核心被 BH 整套收購**——一個由 100+ 個獨立工業企業組成的集團,展示 BH「**收購整個集團 + 給對人就放手**」的能力。

## 投資/收購大事年表
| 年份 | 事件 |
|------|------|
| **2008** | BH 收購 60%,投入約 45 億美元(Pritzker 家族遺產分配的契機) |
| 2010-2014 | 分階段收購剩餘股權 |
| 2014 | **完全收購完成** |

## 為什麼這筆交易很特別

### 一、規模龐大但低調
馬蒙旗下有 100+ 家獨立子公司,涉及鐵路設備、電線、水管、消費品工業等——
**沒有任何單一品牌出名,但加起來規模驚人**。

### 二、Pritzker 家族的信任
Jay Pritzker 與巴菲特長期友誼。家族遺產分配時,**選擇 BH 作為延續性的買家**——
這是「**信譽複利**」的一個具體展現:**好名聲讓 BH 經常成為**家族企業出售時的首選**。

### 三、無縫接管
BH 收購後**幾乎沒做改變**——保留所有子公司管理層、保留分散結構。
這是 [[經理人選擇]] + [[資本配置]] 紀律的展示。
