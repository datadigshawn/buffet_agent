# 樂購(Tesco)

## 一句話定位
**英國最大連鎖超市**——BH 在 2007 開始建倉,但 2014 會計醜聞曝光後**全部出清**,損失約 4.44 億美元。

## 投資/收購大事年表
| 年份 | 事件 |
|------|------|
| 2007 | BH 開始建倉 |
| 2012 | 持股約 5% |
| **2014/9** | **Tesco 會計醜聞曝光**——管理層誇大利潤約 2.6 億英鎊 |
| 2014 後 | BH 完全出清,損失 4.44 億美元 |

## 為什麼出清

### 一、會計造假是不可接受的
[[誠信與品格]] 概念在這裡成為**否決票**:
- 巴菲特公開說(語意改寫):**「**會計造假是 BH 永遠不能接受的——即使其他面向都好,**我們也會出清**。**」**

### 二、巴菲特的「**自我警告**」
他公開檢討(語意改寫):
**「**我從 2012 年起就**對 Tesco 有疑慮**——但我**沒有及時行動**,讓持股繼續滾大。這是個值得記取的教訓:**疑慮就應該行動**。**」**

### 三、雙重失誤
- **作為失誤**:當初進場 + 沒及時退場
- **教訓深度**:對「**疑慮但不行動**」的反思
