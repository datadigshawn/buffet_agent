# 水牛城新聞報(Buffalo News)

## 一句話定位
**從 BH「**特許經營權教科書**」到「**護城河被網路填掉**」的最完整縮時演示**——1977 收購、2020 售出,跨度 43 年。

## 投資/收購大事年表
| 年份 | 事件 |
|------|------|
| **1977** | BH 以 3,500 萬美元收購 |
| 1980s | 與當地對手競爭,最終勝出 → **獨佔當地報業** |
| 1980s-2000s | **教科書級的特許經營權**——獨佔市場 + 強定價權 + 高 ROIC |
| 2010s | **網路新聞、Craigslist 衝擊**——傳統報業崩跌 |
| **2020** | BH 將 Buffalo News 與其他報紙資產**售給 Lee Enterprises** |

## 為什麼這個故事這麼重要

### 1977-2000s:特許經營權的活範例
水牛城新聞報長期是巴菲特用來**解釋 [[特許經營權]] 概念**的具體案例:
- **獨佔當地市場**(競爭對手退出後)
- **不可替代的廣告平台**(本地商家別無選擇)
- **高 ROIC**、輕資本

### 2010s:護城河被填的活見證
網路時代後:
- **Craigslist 拿走分類廣告**
- **Google + Facebook 拿走廣告主**
- **訂閱戶逐年流失**

巴菲特公開承認(語意改寫):**「我**低估了網路對報業的破壞速度**。」**

### 對護城河概念的啟示
這個案例對 BH 投資哲學的影響極大——它證明:
- **沒有護城河是永恆的**
- **科技可以填平百年積累的優勢**
- 投資判斷必須**持續更新**
