# 迪士尼(The Walt Disney Company, DIS)

## 一句話定位
**巴菲特一生兩次買賣迪士尼,**兩次都賣得太早**——是他公開承認的「不作為失誤」之一。

## 投資/收購大事年表
| 年份 | 事件 |
|------|------|
| **1966** | **合夥人時期建倉**——以約 4 美元/股買進 5% 迪士尼 |
| 1967 | 賣出獲得約 50% 報酬 → **這之後迪士尼股價漲了上百倍** |
| 1996 | 透過 [[首都_ABC|首都/ABC]] 合併再次取得迪士尼持股 |
| 2000 前後 | 出清 |

## 兩次「**賣早了**」

### 1966-1967 第一次
巴菲特在合夥人時期以極便宜的價格買進迪士尼——
**那時迪士尼公園的概念剛開始發力**:
- Walt Disney World 即將開幕
- 迪士尼動畫帝國尚未真正爆發
- IP 庫(米奇、白雪公主、灰姑娘等)正在累積複利

巴菲特賣掉以後,迪士尼股價在 1970s-80s 漲了**上百倍**。

### 1996-2000 第二次
透過併購取得後,BH 在 2000 年前後出清——
這之後迪士尼又經歷皮克斯、漫威、星戰等收購,**估值再翻數倍**。

## 巴菲特公開檢討

巴菲特多次自嘲(語意改寫):**「**迪士尼是我最會買、但也最會賣早**的公司——**證明我有時候**真的不認得**自己手上有什麼**。」**

這是 [[失誤與認錯]] 概念的具體應用——
**不作為(該繼續持有沒持有)的失誤,代價極大**。
